How it works

  1. The form posts the selected department code and year to /methodology/phase-5.
  2. The Flask handler validates that both values are in the set the model saw during training.
  3. The input is one-hot encoded the same way as during training and scaled with the persisted StandardScaler.
  4. The trained LogisticRegression model returns a probability between 0 and 1 for the High risk class.
  5. The page renders the predicted class, the probability, and a coloured indicator bar.
